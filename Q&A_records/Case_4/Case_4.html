Phase 2. CONSTRAINT ANALYSIS

  1. All operations must respect the logical predecessor sequence. Each operation cannot start until all its prerequisites have finished.
  2. The first two operations in the entire plan must be “S40_00001_Jig in” and “S40_01001_Set up working environment”; the last operation must be “S40_00002_Jig out.”
  3. The operation “S40_04012_Deburring int, positioning, attach them automatic” or “S40_04013_Deburring int, positioning, attach them manual” is the final step that completes one 1/4 fuselage joint. Predecessors (such as drilling, riveting) must be done beforehand.
  4. Automatic joint operations can only be performed in series for each 1/4 body, and only one 1/4 body can be automatically jointed at a time.
  5. Manual joint operations can be performed in series or in parallel, up to two bodies in parallel, but only if they are executing the same manual operation.
  6. Manual and automatic operations cannot run at the same time in parallel.
  7. After “S40_04014_Deinstall LFT and rails” (auto final) or “S40_04013_Deburring int, positioning, attach them manual” (manual final), the plan must include “S40_02002_Cleanup and add sealant” followed by “S40_02003_Inspection.”
  8. If multiple 1/4 bodies are manually jointed (in series or parallel), then “S40_02002_Cleanup and add sealant” and “S40_02003_Inspection” are only needed once, after all those manual joints complete.
  9. If multiple 1/4 bodies are automatically jointed in series, we only need one “S40_02001_Set in position Rails and LFT” performed first (before all auto joints), and one “S40_04014_Deinstall LFT and rails” performed last (after all auto joints).

Automatic vs. Manual Quarter Joint Logic:
• Automatic path (for a 1/4 fuselage) typically includes “S40_02001_Set in position…” → drilling/riveting → ends with “S40_04012_Deburring int, positioning, attach them automatic” → “S40_04014_Deinstall LFT and rails” → then cleanup & inspection.
• Manual path (for a 1/4 fuselage) typically includes drilling template installation, manual drilling, manual riveting, with the final “S40_04013_Deburring int, positioning, attach them manual” → then cleanup & inspection.
• The plan must incorporate exactly 4 quarter fuselages, mixing some automatic and some manual, subject to parallel/series constraints.
